# kb-argaman

Репозиторий состоит из двух частей:

- `kb/` — база знаний (контент), который потом используется для векторизации и/или публикации.
- `site/` — сайт на Docusaurus для просмотра контента.

Документация по работе с репозиторием:

- Правила и договорённости: `CONTRIBUTING.md`
- Инструкции для Codex/агентов: `AGENTS.md`
