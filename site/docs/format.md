# База знаний (векторная)

Эта папка — хранилище коротких текстов, которые позже будут превращены в векторы.

## Формат записи
- Один файл = одна тема.
- Язык контента: иврит.
- В начале файла — YAML-метаданные.

Пример структуры:

```
---
id: kb-001
title: Заголовок
language: he
tags: [тема1, тема2]
source: "как есть"
---

Текст на иврите...
```

## Как будем наполнять
Я буду задавать простые вопросы. Ты отвечаешь коротко, а я превращаю это в новые файлы.
